# 👁️ Анти-ИИ Зарядка для глаз

Бесплатный офлайн-тренажёр для здоровья глаз. Разработан для борьбы с цифровым утомлением и профилактики «компьютерного зрения».

## ✨ Особенности

- **Полностью офлайн** — работает без интернета
- **Без рекламы** — навсегда бесплатно
- **Без сбора данных** — ваша приватность под защитой
- **5 уроков** с прогрессией от простого к сложному
- **Голосовое сопровождение** — не нужно читать инструкции
- **Анимированные упражнения** — глазодвигательная гимнастика

## 🎯 Упражнения

- Шторки (частое моргание)
- Точка (слежение за движущимся объектом)
- Геометрия (рисование фигур глазами)
- Жмурки (интенсивное сжатие век)
- Пальминг (расслабление с закрытыми глазами)
- Паузы для отдыха

## 📱 Как использовать

1. Откройте приложение
2. Нажмите «Начать зарядку»
3. Следуйте голосовым подсказкам
4. Занимайтесь 1-2 раза в день
5. Результат заметен через 2-3 недели

## 🛠 Технологии

- HTML5 / CSS3
- Vanilla JavaScript
- Canvas API для анимации
- Web Audio API

Приложение ожидает аудиофайлы в папке `audio/`:

| Урок | Файлы |

| Урок 1 | `01_lesson1_intro.mp3`, `01_1_blink_*.mp3`, `01_2_ball_*.mp3`, `01_3_rest_*.mp3`, `01_lesson_outro.mp3` |
| Урок 2 | `02_lesson2_intro.mp3`, `02_1_geometry_*.mp3`, `02_2_ball_*.mp3`, `02_3_rest_*.mp3`, `02_lesson_outro.mp3` |
| Урок 3 | `03_lesson3_intro.mp3`, `03_1_squeeze_*.mp3`, `03_2_ball_*.mp3`, `03_3_rest_*.mp3`, `03_lesson_outro.mp3` |

## 🚀 Быстрый старт

1. Клонируйте репозиторий
2. Добавьте свои аудиофайлы в папку `audio/`
3. Добавьте `frame.jpg` и `cube.mp4` в корень
4. Откройте `index.html` в браузере
